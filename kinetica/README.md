# Kinetica

**Professional motion graphics toolkit for Blender**
By **Saeed Amani**

Compatible with Blender 3.x · 4.x · 5.x  (full Slotted Actions support)

## What it does

15 production-ready motion graphics presets that you fire onto any
selected object — meshes, empties, armatures, cameras, anything that
moves. All presets use *delta* transform channels, so they layer on
top of your existing animation without ever touching it.

## Installation

1. Edit → Preferences → Add-ons → Install…
2. Pick `kinetica.zip` (do **not** unzip first)
3. Enable "Kinetica"
4. Open the N-panel in the 3D Viewport → **Kinetica** tab

## Presets

**Fade**            Fade In · Fade Out (driver-based, real-time duration)
**Scale**           Pop In · Pop Out · Bounce In · Squash · Drop & Roll · Zoom Punch
**Move**            Move In · Move Out (6 directions)
**Rotation**        One-shot rotation on X / Y / Z, custom turns, CW or CCW
**Continuous**      Float · Spin · Wobble · Shake · Pulse · Swing
**Cleanup**         Remove Fade · Refresh Drivers · Clear Transforms · Clear EVERYTHING

## Power features

- **Stagger** — multi-select objects, set Stagger N → each starts N frames after the last. Works with every preset.
- **Stack Mode** — layer new presets on top of existing animation in the same channel without erasing earlier keyframes.
- **Loop Cycle** — independent speed control for continuous loops.
- **Auto Single-User** — fading objects that share a material auto-copies the material so only the target fades.
- **Quaternion → XYZ** — auto-converts glb/glTF rotations to Euler before applying rotation presets.
